Activate component + open editor

Create feature branch
git checkout -b feature/your-feature-name
3
Use Claude Code to build

Open PR via GitHub CLI
gh pr create --web
GitHub App Setup
A
Install Claude Code + GitHub CLI
npm install -g @anthropic-ai/claude-code && gh auth login
Node.js 18+ required · gh auth opens browser OAuth flow
B
Install GitHub App (inside claude session)
/install-github-app
Grants Read + Write access to Contents, Issues, Pull Requests · enables @claude mentions on issues
C
Create CLAUDE.md at repo root
claude "Generate a CLAUDE.md for k9-orchestrator with coding standards, venv activation, and common commands"
CLAUDE.md gives Claude persistent context on every session — treat it like a team onboarding doc
D
Run in tmux for persistent remote sessions
tmux new -s k9-claude && claude
Moonlight/SSH drops won't kill your Claude session · reattach with: tmux attach -t k9-claude
E
Auto-approve safe commands
/permissions
Set auto-approve for git status, ls, pytest — reduces manual taps when working from Kindle
